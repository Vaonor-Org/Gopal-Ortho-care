## Master Design Prompt

You are building a premium hospital website for **Gopal Ortho**, an orthopaedic and physiotherapy centre in Nagercoil, Tamil Nadu, India.

The design language is **Medical Precision Luxury** — think Hermès-level restraint applied to bone and joint medicine. The site must feel like the most expensive, most trusted specialist clinic in South Tamil Nadu. Every pixel must communicate clinical authority and human warmth simultaneously.

---

## The Non-Negotiable Design Rules

### 1. Never Use Emoji
Every icon must be a real SVG. No Unicode emoji anywhere in any file, ever. Use Lucide Icons (stroke-based) for UI elements and custom inline SVGs for medical anatomy icons.

### 2. The Colour Religion
There are only two accent families:
- **Deep forest green** `#0B3D2E` — trust, authority, medicine
- **Warm gold** `#C8973A` — premium, warmth, trust signals

Never use generic hospital blue (`#0077B6` or similar). This is the single biggest visual mistake to avoid.

### 3. Typography is Sacred
- `Cormorant Garamond` — every heading, every display text, every hero word
- `DM Sans` — every body paragraph, every nav link, every UI label
- `DM Mono` — numbers and stats ONLY (e.g., "500+", "15+")
- Mixing these incorrectly immediately destroys the luxury feel

### 4. Animation Philosophy
Every motion must feel like precision medical instruments. No bounce, no wobble, no spring overshoot except where explicitly specified. Think: MRI machine movement, ECG rhythm, surgical robot precision.

All animations must be in `@media (prefers-reduced-motion: no-preference)`.

### Scroll Reveal (via IntersectionObserver)
```javascript
// Apply to any element that should animate on scroll
const observer = new IntersectionObserver((entries) => {
  entries.forEach(entry => {
    if (entry.isIntersecting) {
      entry.target.classList.add('revealed');
      observer.unobserve(entry.target);
    }
  });
}, { threshold: 0.12 });

// CSS
.reveal {
  opacity: 0;
  transform: translateY(32px);
  transition: opacity 700ms cubic-bezier(0.16, 1, 0.3, 1),
              transform 700ms cubic-bezier(0.16, 1, 0.3, 1);
}
.reveal.revealed { opacity: 1; transform: translateY(0); }

// Stagger children: add --delay CSS variable
.reveal-child { transition-delay: var(--delay, 0ms); }
/* Set --delay: 120ms, 240ms, 360ms etc on each child */
```

## Stats Count-Up (JavaScript)
```javascript
function countUp(el, target, duration = 2200) {
  const start = performance.now();
  const easeOutExpo = t => t === 1 ? 1 : 1 - Math.pow(2, -10 * t);
  
  function update(now) {
    const elapsed = now - start;
    const progress = Math.min(elapsed / duration, 1);
    const value = Math.floor(easeOutExpo(progress) * target);
    el.textContent = value.toLocaleString() + '+';
    if (progress < 1) requestAnimationFrame(update);
  }
  requestAnimationFrame(update);
}

const statsObserver = new IntersectionObserver((entries) => {
  entries.forEach(entry => {
    if (entry.isIntersecting) {
      const target = parseInt(entry.target.dataset.target);
      countUp(entry.target, target);
      statsObserver.unobserve(entry.target);
    }
  });
}, { threshold: 0.3 });

document.querySelectorAll('[data-countup]').forEach(el => statsObserver.observe(el));
```

## Booking Form Step Transitions (JavaScript)
```javascript
function goToStep(from, to) {
  const fromEl = document.getElementById(`step-${from}`);
  const toEl = document.getElementById(`step-${to}`);

  // Exit current step
  fromEl.style.transition = 'transform 0.35s ease-in, opacity 0.35s ease-in';
  fromEl.style.transform = 'translateX(-60px)';
  fromEl.style.opacity = '0';

  setTimeout(() => {
    fromEl.style.display = 'none';
    toEl.style.display = 'block';
    toEl.style.transform = 'translateX(60px)';
    toEl.style.opacity = '0';

    // Force reflow
    toEl.offsetHeight;

    // Enter new step
    toEl.style.transition = 'transform 0.4s ease-out, opacity 0.4s ease-out';
    toEl.style.transform = 'translateX(0)';
    toEl.style.opacity = '1';

    // Update progress bar
    updateProgressBar(to);
  }, 350);
}
```
